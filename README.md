# 👨‍💻 AlexVila04 - Developer Portfolio

A highly interactive and stylized developer portfolio built with **Next.js 16**, **React**, **Tailwind CSS v4**, and **Framer Motion**. It features a retro terminal/IDE boot sequence Entrance page that transitions into a sleek, modern, dark-themed portfolio showcasing projects, server experience, and skills.

## ✨ Features

- **Retro Entrance Sequence:** Simulated runtime boot process with a custom IDE file explorer UI.
- **Interactive Animations:** Smooth scroll transitions, hover effects, and parallax elements powered by `motion/react`.
- **Custom Theming:** A tailored dark mode aesthetic with deep greens, golds, and blues, utilizing Tailwind v4 variables.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Sectioned Layout:** 
  - **Hero:** Quick intro with stats (Plugins, Servers, Projects)
  - **Projects:** Open source, university, and plugin showcase
  - **Servers:** Highlighting experience running and managing Minecraft networks
  - **Current Work:** "Coming Soon" placeholder for active build phases
  - **Featured In:** Highlighting community impact (e.g., YouTube showcases)
  - **Stack & About:** Technical skills and personal background as a Systems Engineering student

## 🛠️ Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Motion](https://motion.dev/) (Framer Motion)
- **Icons:** [Lucide React](https://lucide.dev/) & Custom SVGs
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AlexVila0204/portfolio-nextjs.git
   cd portfolio-nextjs
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the entrance sequence, or go directly to [http://localhost:3000/portfolio](http://localhost:3000/portfolio) for the main portfolio page.
